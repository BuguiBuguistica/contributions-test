### Getting started

1. Install angular-cli (last version)

npm install -g @angular/cli

If you have old version upgrate it:

npm uninstall -g angular-cli
npm cache clean
npm install -g @angular/cli@latest

2. Install node modules

npm install

3. Start application

npm start
